# Quick Guide: Save Your Images

Please save the images you uploaded into these exact locations:

## Step 1: Create the folder structure (if not exists)
The folder `src/assets/hero/` should already exist.

## Step 2: Save the images with these exact names:

1. **First image (large printer)** → `digital-printer-1.jpg`
2. **Second image (colorful offset printer)** → `offset-printer.jpg`
3. **Third image (printer close-up)** → `digital-printer-2.jpg`
4. **Fourth image (printer with colored sheets)** → `offset-samples.jpg`
5. **Fifth image (brochures on table)** → `brochures-poster.jpg`

## File locations:
```
src/assets/hero/digital-printer-1.jpg
src/assets/hero/offset-printer.jpg
src/assets/hero/digital-printer-2.jpg
src/assets/hero/offset-samples.jpg
src/assets/hero/brochures-poster.jpg
```

## After saving:
The images will automatically appear in your hero section!
Just refresh the page (Ctrl+Shift+R) if needed.
